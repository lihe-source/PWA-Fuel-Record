ℹ️ 如何設定 Google API？ ▼

1. 前往 Google Cloud Console

2. 建立新專案或選擇現有專案

3. 啟用 Google Drive API

4. 到「憑證」頁面，建立 OAuth 2.0 用戶端 ID（類型選 Web 應用程式）

5. 在「已授權的 JavaScript 來源」加入您的網站網址（例如：https://lihe-source.github.io）

6. 複製 Client ID，貼至上方欄位後登入
